## About Interface
- 인터페이스에 언언된 프로퍼티 또는 메소드의 구현을 강제하여 구조적으로 견고하게 해준다.
- 어플리케이션에 필요한 커스텀 타입 추가가 가능하다.
- ES6는 인터페이스를 지원하지 않지만 TypeScript는 인터페이스를 지원한다.

제네릭은 선언 시점이 아니라 생성 시점에 타입을 명시하여 하나의 타입만이 아닌 다양한 타입을 사용할 수 있도록 하는 기법이다.
